But how do I get reputation in Python specifically?

  1. Lots of Unicode questions - have a pre-rolled answer
  2. Lots of homework questions - learn to answer simply
  3. Don't answer NumPy questions - there is a better way you don't know about
  4. Answer Django questions - high question to answer ratio
  5. Learn collections and itertools - replace tons of custom code
    http://docs.python.org/library/collections.html
    http://docs.python.org/library/itertools.html
  6. Use big-O notation - people like fast
    http://wiki.python.org/moin/TimeComplexity
  7. Read "Python Datamodel" and the CPython source - learn the internals
    http://docs.python.org/reference/datamodel.html
